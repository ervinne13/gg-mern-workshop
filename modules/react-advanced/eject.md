# Ejecting From CRA

CRA or create react app allows us to easily create React based applications with almost no configuration. It will also automatically setup a webpack dev server for better development experience.

"Ejecting" from CRA means we "expose" the configuration that CRA is hiding from us, letting us configure webpack ourselves.

Follow along the instructor as he walks you through the complex webpack configuration that CRA is handling for us.

To better appreciate this, let's first learn how to use webpack in the first place and be able to make use of it in our non React projects or be able to understand and make configurations to an ejected CRA application.
